# Simple API Server

Create a basic HTTP server:

```lua
server = sandman.server.start(8080)

sandman.server.get(server, "/hello", function(req)
    return {
        status = 200,
        body = "Hello, World!",
        headers = { ["content-type"] = "application/json" }
    }
end)
```

Test the endpoint:

```lua
response = sandman.http.get("http://localhost:8080/hello")
print(response.body)
```
